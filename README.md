## Порядок запуска

### Запускаем докер с базой данных

```bash
$ docker-compose up -d
```

Внимание, если БД пустая - проверить директорию ./db-data

### Запускаем сервер

```bash
$ npm run start
```

### Вход в админку БД

http://localhost:8888/

### После закрыть образ докера

```bash
$ docker-compose down
```
